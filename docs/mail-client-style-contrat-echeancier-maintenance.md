# Mail client — Ajustements contrat, échéancier vide et maintenance

**Objet :** Suite à vos demandes — contrat, échéancier et cadre des prochaines évolutions

---

Bonjour,

Nous revenons vers vous suite à vos derniers échanges, en trois points pour gagner en clarté.

**1. Modifications de style du document contrat**  
Les ajustements de mise en forme que vous souhaitez sur le document contrat pourront être traités **dans des délais courts**. Nous sommes actuellement sur une **période de forte charge** ; dès que la fenêtre le permet, nous pourrons les prendre en charge **rapidement**. N’hésitez pas à nous préciser les attentes visuelles ou les exemples utiles pour limiter les allers-retours.

**2. Échéancier vide à l’écran**  
Si vous constatez à nouveau un **échéancier vide** alors que le dossier devrait en afficher un, la manipulation à effectuer est la suivante : dans l’onglet **« Formulaire »**, utilisez le bouton **« Recalculer les échéances »**. Les lignes d’échéances devraient alors **réapparaître**. Ce geste suffit dans les cas où les données n’ont pas été rafraîchies correctement côté interface.

**3. Autres demandes**  
Toute autre évolution ou correction (hors ce type d’ajustement ponctuel déjà évoqué) rentrera dans le **dispositif de maintenance** du site, dans les modalités que nous vous avons communiquées (abonnement ou intervention à la tâche).

Nous restons à votre disposition pour la suite.

Cordialement,
